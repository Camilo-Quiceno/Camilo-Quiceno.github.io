Daubara

3D-Design Programming

As my thesis work, I developed a training module for doctors interested in spine surgery, in partnership with GIBIC (Research Group Bioinstrumentation and Clinical Engineer) For the development of this module, I applied all the knowledge acquired by the group in the simulator DAUBARA.

DAUBARA is a tool which allows preoperative planning and intraoperative navigation. The system includes an electromagnetic navigator and a set of physical models (3D-printed). DAUBARA has been used for third ventriculostomy training during two years.

The module, I contributed with, was developed to train the surgeons to position screws in the spine. As the user positions the screws in the 3d-Printed model, it will be displayed virtually in realtime on the screen of the computer. This technology allows the doctors to practice the placement of the implants before going into real life experience with a patient.

The project uses these technologies: 3DSlicer, Python, 3D-Priting & Electromagnetic navigation.

CEToolKit

Python Programming

In my current Job, a designer must face 3 issues during desing/inspection: time, technical knowledge and KPI registration.

CETooklit was created to collect the solution of these issues. The first tap is in charge of controlling the amount of time that the engineer must consume on the design. Tabs number 2 to 4 keep all the technical information that the user should check during his work and the fifth tab allows tracking all the data related with the performance of the user.

The project uses these technologies: Python, Excel integration.
